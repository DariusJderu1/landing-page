# British Shorthair Landing Page

A static landing page dedicated to British Shorthair cats. This project focuses on structuring HTML documents and applying CSS Flexbox for layout management and element alignment.

**Live Demo:** [https://dariusjderu1.github.io/landing-page/](https://dariusjderu1.github.io/landing-page/)

## Technologies & Tools Used

* **HTML5:** Utilized for semantic markup to structure the landing page sections (navbar, hero, content, quote, call-to-action, footer).
* **CSS3:** Extensive use of CSS Flexbox to align elements, manage spacing (`gap`, `justify-content`, `align-items`), and create an image grid. Google Fonts (Roboto) is integrated for typography.

## Features

* **Hero Section:** A prominent header with a call-to-action button and a main hero image.
* **Information Grid:** A flexbox layout displaying different types of British Shorthairs, utilizing `object-fit` to maintain uniform image aspect ratios.
* **Quote Section:** A styled section for a featured quote with specific typographic alignment.
* **Call to Action (CTA):** A styled banner prompting user sign-ups.

## Getting Started

This is a static website and does not require any build tools or dependencies to run locally.
