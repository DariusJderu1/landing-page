# British Shorthair Landing Page

A static, responsive landing page dedicated to British Shorthair cats, built as part of The Odin Project curriculum. The project focuses on structuring HTML semantically and applying CSS Flexbox for layout management.

**Live Demo:** [https://dariusjderu1.github.io/landing-page/](hhttps://dariusjderu1.github.io/landing-page/)

## Technologies & Tools Used

* **HTML5:** Semantic markup to structure the landing page sections (navbar, hero, content, quote, call-to-action, footer).
* **CSS3:** Extensive use of CSS Flexbox to align elements, manage spacing (`gap`, `justify-content`, `align-items`), and create a responsive image grid. External fonts from Google Fonts (Roboto) are integrated for typography.

## Features

* **Hero Section:** A prominent header with a call-to-action button and a main hero image.
* **Information Grid:** A responsive flexbox grid displaying different types of British Shorthairs with uniform image sizing using `object-fit`.
* **Quote Section:** A styled section for a featured quote with specific typographic alignment.
* **Call to Action (CTA):** A styled banner prompting user sign-ups.

## Getting Started

This is a static website and does not require any build tools or dependencies to run locally.
